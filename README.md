#### Java-Demo
**本项目（Demo）分为四个部分**
**一：Java基础Demo：**
主要收集一些Java基础的小案例比如：设计模式；IO；线程.......
**二：Java进阶Demo**
主要收纳javaWeb方面的Demo
**三：Spring框架与常用插件框架整合**
也是本项目比较重要的一部分：主要收纳SSM,Spring,SpringBoot与常用的框架的整合比如：Redis,Shiro,Cas,Java操作MongoDB......
**四：第三方插件Demo**
主要收集比较好的第三方前后端插件的使用方法
